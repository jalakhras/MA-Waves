# المستوى الأول

**الشرح:** القمة/القاع الصغير يتأكّد بـ**قاعدة الشمعتين الصحيحتين** في اتجاه التصحيح (شمعة صحيحة = جسم ≥ كل ذيل، مع تجاهل الابتلاع). + **التتابُع** (دافعة تتجاوز سابقتها) و**الفعالية** (لا تُرسم إلا بعد التأكّد).

**الكود:** `minCorr=2` (25)، `useValid=true` (27)، كشف الصغيرة (~880‑945)، `smallActiveDir` بانقلاب البنية (904‑908)، `updateDeg(memS…)` (945).

|---|---|

**الثغرات:** طفيفة (معايرة `minCorr` تقديرية). **النسبة: 95%.**

## الدرس 06 — القمم والقيعان المتوسطة — **95%**

**الشرح:** المتوسطة تُشتقّ من الصغيرة **بالكسر** (قاعدة الاستراتيجية)؛ الخط على **جسم** الشمعة؛ الفعالية (المُغطّى ⇒ أحمر) والتتابُع.

**الكود:** `feed(pmed…)` (الاشتقاق بالكسر، ~1010)، `levelM = eTopM ? eBHiM : eBLoM` (الجسم)، الفعالية **التزايُدية** (تغطية [M4] + تتابُع [M3]، 1088‑1108)، `medActiveDir` (1037‑1094).

| البند | الحالة |
|---|---|
| الاشتقاق بالكسر | ✅ |
| الخط على الجسم | ✅ |
| الفعالية (تغطية+تتابُع) تزايُدياً non-repainting | ✅ (المرحلة 1) |

**الثغرات:** [M6] «بداية الموجة = صغير+متوسط» اعتُبر تجميلياً (مُتّفق على تخطّيه). **النسبة: 95%.**

**الشرح:** الكبيرة من **آخر 7 موجات**؛ تُشتقّ بالكسر من المتوسطة الفعّالة؛ تُنظَّف بالتتابُع/الفعالية؛ النقطة الحدودية تُثبَّت.

**الكود:** استخراج `kP/kB` (1700‑1730)، مساران: **بالكسر** (`useBigBreak`) أو **legs**؛ حدّ آخر 7 قبل `histLeft`؛ `bluePivotBars` (1727‑1730).

| البند | الحالة |
|---|---|
| الاستخراج + التنظيف + آخر 7 + الحدودية | ✅ |

**الثغرات:** 🟠 المسار الافتراضي **ليس بالكسر** (تفعيل `useBigBreak` يرفعها). **النسبة: 85%.**

## الدرس 08 — خطوط السيولة + مستويات الاتجاه — **95%**

**الشرح:** خط السيولة = آخر **قاعين** (صعوداً)/**قمتين** (هبوطاً)؛ مستوى الاتجاه = آخر قاع/قمة؛ **على آخر موجة نشطة فقط، لا تاريخ التاريخ**؛ كسر نصف الجسم.

| البند | الحالة |

| الموجة النشطة فقط (لا تاريخ التاريخ) | ✅ (بوّابة waveStart) |
| منع التكدّس (clamp ضمن النطاق) | ✅ |

**الثغرات:** طفيفة. **النسبة: 95%.**

# المستوى الثاني

## الدرس 01 — بنية المستوى الثاني (3 مراحل) — **90%**

**الشرح:** كل مستوى = **الرسم ← فرص الدخول ← شروط الانعكاس**.

**الكود:** الطبقات الثلاث حاضرة (المناطق=الرسم، `opp*`=الفرص، `entry*`=الانعكاس).

**الثغرات:** لا قاعدة ميكانيكية مفردة (تعريفيّ). **النسبة: 90%.**

## الدرس 02 — رسم المناطق — **85%**

**الشرح:** خط m.A على جسم كل قمة/قاع متوسط فعّال (والزرقاء)؛ مستطيل بين مستويين **قريبين**، لا عبر تصحيح متجاور؛ المنفردة ⇒ تاريخ التاريخ؛ زوج واحد لكل نقطة.

**الكود:** خوارزمية islast (greedy عالمي + `usedZ` + `f_zoneContain`)، عضوية `inZone` per-bar (تُبنى عند الانبعاث)، `f_effForZone` (override الزرقاء)، الجوار **الفعّال** (لا index).

| البند | الحالة |
|---|---|
| m.A على الأجسام الفعّالة (+زرقاء override) | ✅ |
| الجوار الفعّال + الزوج الواحد + منع الاحتواء | ✅ (الخطأ القديم مُصلَح) |
| المنفردة ⇒ تاريخ التاريخ + m.A فقط | ✅ |
| **رسم المستطيلات** | ⚠️ `zoneAutoBox=false` (خطوط m.A فقط) |
| **سقف عرض المنطقة** | ⚠️ نوقش ثم تُرك (مناطق عريضة ممكنة) |

**الثغرات:** 🟠 المستطيلات مطفأة افتراضياً؛ ولا سقف عرض (مشتقّ من سعة الصغيرة) بعد. **النسبة: 85%.**

## الدرس 03 — فرص الدخول — **70%**

**الشرح:** أساسية (منطقة + اتجاه + تجميع/تصريف) + ثانوية عالية المخاطر (تصحيح بعد الزعر)؛ درجة الصفقة = درجة التصحيح؛ صفقات متكرّرة؛ الهدف = المنطقة التالية.

**الكود:** `oppBasicBuy = inZone and buyOK and mc2==3` (تجميع)، `oppBasicSell …==4` (دمج م4+م5)، `oppHR*` (ثانوية)، `pendBuy/Sell Med/Small`، إلغاء pending.

| البند | الحالة |
|---|---|
| أساسي = تجميع/تصريف (تصنيف الدور بالمرحلة) | ✅ |
| درجة الصفقة (صغير/متوسط) | ✅ |
| ثانوية (تصحيح بعد الزعر) | ✅ منطقياً (لكن «؟» فقط) |
| **توليد الإشارة (الأخضر)** | 🔴 **نادر/معطّل** (pending يُمسح قبل الكسر — قيد الإصلاح v1.63→64) |
| **الثانوية بلون مميّز** | ❌ غير منفّذ (تظهر «؟») |

**الثغرات:** 🔴 السلسلة لا تُنتج إشارات مؤكّدة (الجذر: `not buyOK` يمسح pending)؛ الثانوية بلا لون مميّز. **النسبة: 70%.**
