## النمط القابل للتكرار `*ExtrDir`
لكلّ درجة D فوق درجة أصغر d:
1. `var int DExtrDir = 0`.
2. **قبل** استخراج D (داخل `if cfHas` نظيرتها): نقطة d فعّالة تكسر بنية D بالجسم ⇒ اقلب `DExtrDir`:
   - قمّة d جسمها > `memD.ltP` ⇒ `+1` · قاع d جسمه < `memD.lbP` ⇒ `-1` · تمهيد `largeDir?:bigDir` عند الصفر.
3. استخراج D يقرأ `DExtrDir` فقط (`wdir`). **آثار D الجانبيّة (الإرساء/مسح الخطوط/قياد الأصغر/L09-10) بلا مساس.**
