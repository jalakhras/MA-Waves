## 3. القرارات المعتمَدة (ليست فجوات — مقصودة وموثَّقة)

1. **الوقف = حافّة منطقة الدخول** (لا أسفل آخر قاع) — قرار المستخدم.
2. **قوّة السيولة من الصغيرة** باتجاه الترند الأكبر («عكس الصغيرة» وقت الانقلاب).
3. **`keepBroken = false`** (المستوى ينتقل، السيولة تُمسح — حرفيّ الدرس 09).
4. **لا دخول بلا منطقة هدف** (تفسير محافظ أمين).
5. **ق3 على رسم الصغيرة تصويريّ** (التغذية للمتوسطة خام).
6. **الكسر البنيويّ شمعتان فقط** (بلا شرط جسم/ذيل — حرفيّ).
7. **استخراج الكبيرة بالكسر = الافتراضيّ**.
8. **المنفرد (m.A) = هدف لا منطقة دخول** (v3.87.13).
9. **رسم خطّ السيولة بلا شرط تصاعد** (الدرس 08 حرفيًّا، v3.87.14) — حارس «غير مكسورة» + التبويب ببار الكسر يبقيان.
10. **swingMaxDays = 10** معتمَد.

## 4. حدود Pine المتأصّلة (لا تُصلَح برمجيًّا)

- اختيار الموجة / تبديل الفريم / التكبير–التصغير = **بصريّ/يدويّ** (لوحة اختبار الموجة ترشد).
- `hohMinRatio = 0.618` = تقريب فيبوناتشي لقاعدة بصريّة (مُدخَل قابل للضبط).

## 5. بند اختياريّ مفتوح (ليس فجوة)

- إزالة شرط التصاعد من **إسقاطات الذعر** (`medLiqLvl`/`smLiqLvl`) أيضًا للاتّساق الكامل مع الخطوط — مؤجَّل بقرار المستخدم (يرفع حساسيّة الذعر).
